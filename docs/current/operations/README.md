# Operations

Operational references for live hosts, release infrastructure, and node
administration that are part of the current Koinos One / `teleno_node` work.

- `SERVER_INVENTORY.md` - private local-only server inventory. This file is
  intentionally ignored by Git because it can contain hostnames, IPs, SSH users,
  workloads, and resource snapshots.
- `PRODNET_OBSERVER_DOCKER_DEPLOYMENT.md` - current Docker-based prodnet
  observer deployment, public-bootstrap restore, verification, and sizing notes.
