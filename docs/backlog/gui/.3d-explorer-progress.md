## Decisions

- Pinned @react-three/fiber to v8 line (v9 requires React 19; app is on React
  18.3).
- Placeholder scene uses `frameloop="demand"` when hidden or reduced-motion
  instead of unmounting, so the WebGL context survives tab flips.
- `/// <reference types="vite/client" />` added in Explorer3DView for
  `import.meta.env.DEV` (tsconfig does not include vite/client globally).
