- [x] Phase 0 Spike — deps (three 0.169, @react-three/fiber 8.x, drei 9.x for
  React 18), lazy-loaded placeholder scene behind Explorer "3D" subtab with
  Experimental badge, WebGL2 fallback card, pause on hidden/reduced-motion,
  dev FPS probe, EN/ES i18n. Renderer bundle: main chunk ~unchanged (757 kB /
  207 kB gz), lazy chunk Explorer3DView 822 kB / 220 kB gz loaded on demand.

## Decisions

- Pinned @react-three/fiber to v8 line (v9 requires React 19; app is on React
  18.3).
- Placeholder scene uses `frameloop="demand"` when hidden or reduced-motion
  instead of unmounting, so the WebGL context survives tab flips.
- `/// <reference types="vite/client" />` added in Explorer3DView for
  `import.meta.env.DEV` (tsconfig does not include vite/client globally).
